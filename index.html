  • ¿Qué requiere mi computadora para un sistema de PC? Windows 7 (Windows XP ya no es soportado, aún así podría adaptarme a ese sistema operativo), 2 GB de memoria RAM (por todo lo que normalmente ejecutas al usar tu computadora, además, actualmente los equipos vienen con ás memoria RAM), un disco duro de 500 GB (normalmente todos los componentes necesarios para ejecutar la aplicación ya existen en Windows 7, Si es XP, podríamos requerir hasta 600 MB de espacio). El sistema en sí es pequeño, al rededor de 50 MB. También hay que considerar si requieres una base de datos o no.
